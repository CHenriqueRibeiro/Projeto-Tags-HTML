CONTEUDOS DO MODULO


Para que serve as tags h1-h6?

As tags de cabeçalho são responsáveis por definir o cabeçalho principal de uma página (h1) , assim como os subcabeçalhos, ou subtítulos, (h1 – h6 ) de outras seções de conteúdo.

Para que serve a tag p?

A tag p é utilizada para demarcar um parágrafo.

OBS: Nao usar tags como h1-h6 para escrever paragrafos.

Tag strong strong ou b ?

Quando você usa STRONG , você dá um significado para aquela parte especifica do texto. Você diz que ela deve ser lida com mais força, com mais intensidade. Já com o elemento B você apenas marca em negrito o texto, e só.

  1. Para questão de importancia maior é melhor usar a Tag Strong.

  2. Se for só para deixar em negrito sem nunhuma importancia pode ser usada a Tag B sem problema.
